Introduction

The SocialConnect API allows you to integrate our platform's features into your applications. Our RESTful API provides access to user data, posts, comments, and more.

Base URL

https://api.socialconnect.com/v1

Response Format

All responses are returned in JSON format with the following structure:

{
    "success": true,
    "data": {
        // Response data
    },
    "error": null
}

Authentication

To access the API, you'll need to authenticate your requests using an API key. You can obtain an API key from your developer dashboard.

Using API Keys

Include your API key in the Authorization header of your requests:

Authorization: Bearer YOUR_API_KEY

OAuth 2.0

For user-specific data, you'll need to implement OAuth 2.0 authentication. This allows your application to access user data with their permission.

Users Endpoint

Access and manage user data through our Users API.

GET /users/{user_id}

Retrieve information about a specific user.

Parameters

Parameter Type Description
user_id required string The ID of the user to retrieve
fields string[] Specific fields to include in the response

Example Response

{
    "success": true,
    "data": {
        "id": "123",
        "username": "johndoe",
        "name": "John Doe",
        "email": "john@example.com",
        "profile_picture": "https://..."
    },
    "error": null
}

Posts Endpoint

Create, read, update, and delete posts through our Posts API.

POST /posts

Create a new post.

Parameters

Parameter Type Description
content required string The content of the post
media file[] Media files to attach to the post
privacy string Privacy settings for the post
